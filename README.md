# Medical Store Web App

A React-based web application for managing a medical store.

## Features
- Product listing
- Add/remove items from cart
- Per-item quantity management
- Dynamic cart updates
- Total price calculation
- Simulated payment system
- Cart persistence using localStorage

## Tech Stack
- React.js
- JavaScript
- HTML/CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
